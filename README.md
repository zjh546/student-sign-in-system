# Student Sign-In System

## 项目概述

该仓库旨在记录**面向对象框架技术及应用**课程项目。



## 项目描述

这个项目是一个**签到系统**，包含**服务端**（控制台）和**客户端**（桌面端）的代码。签到发起者可以方便地发起签到、查看签到记录，以及进行数据导入和导出等功能。用户可以通过签到发起者提供的签到码进行签到，查看自己的签到记录。



## 项目特点

### 服务端（控制台）

1. **Node.js 多线程处理**：
   - 采用 Node.js 的多线程技术，包括 worker 和 spawn，以提高服务端性能和并发处理能力。
2. **控制台美化和图像输出**：
   - 实现控制台界面的美化，增强用户体验。
   - 能够在控制台中输出照片，为用户提供直观的信息展示。
3. **Node.js 调用 Java 程序**：
   - 利用 Node.js 与 Java 的交互，实现服务端与 Java 程序的协同工作，拓展系统功能。
4. **局域网服务搭建（Express）**：
   - 使用 Express 框架搭建局域网服务，以支持在局域网内部提供服务端功能。



### 客户端（桌面端）

1. **Electron + Vue + Vite + TS 技术栈**：
   - 借助 Electron 框架、Vue 框架、Vite 构建工具和 TypeScript，构建出高效、响应迅速的桌面端应用。
2. **Electron 的 IPC 通信**：
   - 利用 Electron 的 IPC（进程间通信）功能，实现客户端内部不同进程之间的有效沟通，以提高整体系统的协同性。



## 项目使用

### 客户端

```bash
cd desktop
pnpm i
npm run dev  # 运行
npm run build  # 构建
```



### 服务端

```bash
cd server
pnpm i
npm run dev:main # 运行
```
